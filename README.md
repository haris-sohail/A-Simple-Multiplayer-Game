# A-Simple-Multiplayer-Game

Runs in Ubuntu. Built in C++ using SFML library. Command for running the game: <br><br>
g++ main.cpp -g -o mygame -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio -pthread
